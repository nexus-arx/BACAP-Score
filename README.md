# BACAP Score Tracker (Weighted Advancement Points)

A lightweight, highly optimized companion datapack designed specifically for **BlazeandCave's Advancements Pack (BACAP)**. 

Instead of tracking your progression by a flat count of total advancements completed, this pack introduces a dynamic, **weighted scoring system** where harder, multi-stage achievements grant significantly more points than entry-level milestones.

## 🔗 Modrinth Project Page
You can find the official releases, submit ratings, and view version histories over on the [Modrinth Project Page](https://modrinth.com/datapack/bacap_score).

## 🌟 Features
* **Weighted Points Matrix:** Awards points dynamically (ranging from 1 to 5+ points) based on true advancement difficulty metrics.
* **Highly Optimized Clock:** Evaluates online player data exactly once every 20 ticks (1 second) to conserve server resources and maintain peak server TPS performance.
* **Vanilla Leaderboard Layouts:** Automatically initializes global objectives and configures native Minecraft `below_name` and `list` (Tab menu) scoreboard displays.

## 🛠️ Requirements & Compatibility
* **Minecraft Version:** `26.1`, `26.1.1`, and `26.1.2` (Pack Format 101)
* **Required Dependency:** [BlazeandCave's Advancements Pack](https://modrinth.com/datapack/blazeandcaves-advancements-pack). This utility will not perform actions in a vanilla world without BACAP active.

## 📥 Installation
1. Download the latest release `.zip` file.
2. Drop it directly into your world's `datapacks` directory.
3. Join the world or run `/reload` in the console/chat.
